## Requirements:
1. Nodejs>=16
2. Git
3. A Webhook url
## Installing:
```
git clone https://github.com/HELLSNAKES/neko-hook.git
cd neko-hook
npm i
```
Edit `config.json` and then `npm start`.
> Note: `timeout` unit is millisecond
